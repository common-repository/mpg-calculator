Add a MPG (Miles per gallon) calculator & Fuel Consumption Calculator to your site
 
== Description ==
 
Add a MPG (Miles per gallon) calculator and Fuel Consumption Calculator to your site using 2 separate shortcodes.
 
== Installation ==
 
1. Upload 'mpg-calculator.zip' to the '/wp-content/plugins/' directory
2. Extract the 'mpg-calculator.zip' folder
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Add the MPG Calculator using [mpg_calculator]
5. Add the Fuel Consumption Calculator using [fuel_calculator]
